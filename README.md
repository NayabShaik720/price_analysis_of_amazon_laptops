# Amazon Laptop Price Analysis

## 📌 Project Overview
This project analyzes Amazon laptop pricing data using Python and Jupyter Notebook.

## 🎯 Objectives
- Clean and preprocess the dataset
- Perform Exploratory Data Analysis (EDA)
- Identify pricing trends
- Compare laptop brands and specifications
- Visualize key insights

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📂 Project Files
- laptop_project.ipynb
- laptops_data.csv

## 📊 Key Insights
- Brand-wise price comparison
- Price distribution analysis
- Feature-based pricing trends
- Market insights from Amazon laptop data
